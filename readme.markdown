# 操作

* マウスでお好みの場所をクリック

# 動作

* ちょっとしたエフェクトが発生します
* 動作環境の性能が良ければあちこちで連打しても大丈夫

# 中身

* HTMLのcanvasで2Dコンテキストを使用
* 実はパーティクルベースの簡単な物理シミュレーションをしている
    * 運動方程式、似非動摩擦抵抗
    * オイラー法
* JavaScriptソースは割と綺麗なプロトタイプベースオブジェクト指向
